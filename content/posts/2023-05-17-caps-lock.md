Kate Rose Mosley has a method for [<cite>Detecting caps lock in JavaScript</cite>](https://iamkate.com/code/caps-lock/). ItÂ´s important to listen to `keydown` and `keyup` because browsers behave differently.

```js
addEventListener('keydown', detectCapsLock)
addEventListener('keyup', detectCapsLock)

function detectCapsLock(e) {
  if (e.getModifierState('CapsLock')) {
    console.log('caps lock is on');
  } else {
    console.log('caps lock is off');
  }
}
```
